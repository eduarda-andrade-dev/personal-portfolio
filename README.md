## Tecnologias e Implementação / Technologies and Implementation

O site foi construído utilizando tecnologias fundamentais da web, prezando pela performance e manutenibilidade do código.

* **HTML5:** Estrutura semântica focada em acessibilidade e SEO.
* **CSS3 (Custom Variables & Grid):** Uso de variáveis para gerenciamento de cores e implementação de **CSS Grid/Flexbox** para garantir total responsividade em dispositivos móveis e desktop.
* **JavaScript (DOM):** Implementação de menu interativo (hambúrguer) e lógica de validação para o formulário de contato.
* **GitHub Pages:** Utilizado para hospedagem automática e deploy contínuo.

## Estrutura do Repositório / Repository Structure

- `index.html`: Arquivo principal contendo a estrutura e o conteúdo bilíngue.
- `css/estilos.css`: Folha de estilo centralizada com design responsivo.
- `js/scripts.js`: Lógica de interatividade e manipulação do DOM.
- `img/`: Diretório contendo assets visuais e capturas de tela dos projetos.
- `html/`: Subdiretórios contendo projetos menores hospedados individualmente.
